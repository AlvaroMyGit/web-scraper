#Web Scraper

This is a simple web scraping application built with Java and Spring Boot. It is designed to scrape product information from the Newegg website for both AMD Ryzen and Intel processors.

<h2>Features</h2>

- Scrapes product information for AMD Ryzen processors.
- Scrapes product information for Intel processors.
- Stores scraped data in a PostgreSQL database.


<h2>Technologies Used</h2>

- Java
- Spring Boot
- Selenium WebDriver
- ChromeDriver
- Maven
